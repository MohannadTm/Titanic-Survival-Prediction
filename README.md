# Titanic-Survival-Prediction

This project analyzes the Titanic dataset and builds a machine learning model to predict passenger survival.
Titanic Survival Prediction

Objectives
Clean and preprocess data
Analyze key factors affecting survival
Build a predictive model
Key Insights
Females had higher survival rates than males
First-class passengers had better survival chances
Children had higher survival rates than elderly
Lowest survival: males in third class
Model
Logistic Regression
Accuracy: ~82%
Tools Used
Python
Pandas
Matplotlib
Scikit-learn
How to Run
Install required libraries
Run model.py
